Setting a Titlepage #

To set a title page in LaTeX we have the following command: \maketitle. But we cannot just issue this command by itself. It needs a few argumnentative commands given in order to work. These are: \author{}, \title{} and \date{}. Before we give the \maketitle command we will first have to define the author, title and date by issuing these commands. So the code is:

1\documentclass[a4paper]{article}
2\title{LaTeX personalization}
3\author{writwik ray}
4\date{14-th Nov, 2022}
5\maketitle
6\end{document}

Try and run the code in your machine and play around with changing some parts! Now, you might have noticed that the format of the date is not very nice. It would be better if we could have a superscript after the number, a quick search can give us a nice solution. The solution is in the form of a new package called {nth}, so let’s load up the package and use it! In this particular case we have to load the package with an option (remember option is in [] and package name in {}!) called [super] (signifies that we need the extension in a superscript). The code for loading up the package is as follows, you need to place it at the preamble (between \documentclass and \begin document commands):

1\usepackage[super]{nth}

Now if we want to use the superscript functionality, we simple use the \nth{<number>} command, passing in the number as an argument! Now the updated code is:

 1\documentclass[a4paper]{article}
 2\usepackage[super]{nth}
 3\begin{document}
 4\title{LaTeX personalization}
 5\author{writwik ray}
 6\date{\nth{14} Nov, 2022}
 7\maketitle
 8
 9Hi there! You can see that whatever number we give to the ``\textbackslash nth" command, it produces the output nicely!
10Example: \nth{1} \nth{2} \nth{3} \nth{4} \nth{21}.
11\end{document}

Note: You can simply use the \textsuperscript command to superscript any text, however using this package allows us the mobility of just thinking about the numbers, rather than what extension it would require (‘st’ for 1, ’nd’ for 2 etc.) also, less typing!).

You may have noticed that if we start a new paragraph after the \maketitle command, the paragraph continues in the current page, we might not want that, for that, we can issue the \pagebreak command just after the \maketitle.

Page-Numbering #

Notice in the first page that LaTeX has already started page-numbering without even us telling it to do so! This is all fine and good, but we may not want page-numbers in the titlepage, hence we have to issue a command to tell LaTeX to ignore that page: \thispagestyle{empty}.

By default LaTeX uses arabic numbering(e.g. 1, 2, 3 etc.), but sometimes we might want roman numbering or letters. For that we issue the command: \pagenumbering{roman}. This sets the page numbering in lowercase roman numerals! We have also the following options:

  • alph —– a
  • Alph —– A
  • arabic —– 1
  • roman —– i
  • Roman —– I

Change Font Style #

To change fonts you will need to have one additional package called {fontenc}, which provides proper font encoding. You can go ahead and look for a font for your use over at this link. For our example we are using the anttor style.

We will not cover changing fonts in that much detail since for that we have to use a different compiler (luatex and xelatex). For now you can use trial and error to know which fonts are supported. If you go to the TUG website (link is given above), then you can choose a font and see the configurations you have to make in order to use it.

Change Page Layout #

One of the main problems people face when they are starting out with LaTeX is the page layout. The margins by default are too thick and makes the text-area quiet small. Using the geometry package we can change the page layout to our liking. Say, for example, we want an a4 document with a text-width of 6 inches and height of 8 inches, we can easily do that by: \usepackage[a4paper, total={6in,8in}]{geometry}.

Inserting Pictures (png, jpg etc files) #

As you can see in some of the above examples we have inserted a picture in our document, we do that by loading the graphicx package and giving it the necessary parameters. The proper format is as follows:

1\usepackage{graphicx} %in preamble
2
3\begin{figure}[hbt!]
4	\centering
5	\includegraphics[scale=<number as percentage>]{/path-to-file/file-name}
6	\caption{Caption}
7\end{figure}

The option [hbt!] will keep the picture at the top of the page and relatively at the same place as it is in the actual code, \centering centers the image in relation to the text-width, [scale] does the scaling for us, since most of the times the images are far too large or far too small and finally, \caption{} is used to give the figure a caption that will also be used in the \listoffigures.
