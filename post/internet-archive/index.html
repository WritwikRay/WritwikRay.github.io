Since z-library has been off the grid for a while now (though it is still available in the dark-web, but we don’t want to lurk there), it has been difficult for students, researchers, teachers and all academicians. Access to books is as essential as access to food! As a temporary solution today we’ll be looking into a way how you can access Internet Archive’s books that are usually only available online, offline in pdf format!

Disclaimer: The method used here is for educational uses only, we do not condone copyright violation and infringement. Follow your country’s piracy guidelines and please delete all files after you are done with them!

What’s a Bookmarklet? #

Remember that you can store specific pages as a bookmark on your browser? A bookmarklet utilises this feature of storing pages as a way to store javascript scripts for easier execution. What I mean by this is that you can store javascript code (it has to be all in one line) as a bookmark, and then execute it when you press the bookmark button. Let’s see an example!

Hello world of Bookmarklets #

You can easily create a hello world script using bookmarklet. First navigate to your browser’s bookmark settings and add a custom bookmark. First give it a name, we’ll be going with “Hello World Bookmarklet!”. And as the link of the page, you have to enter the following code:

1javascript:(function()%7Balert(%22Hello%20World%22)%7D)()

What is the above code doing you ask? It is simply opening up an alert box that says “Hello World!”. It’s looking a bit cryptic since we had to properly encode it and write all the JavaScript in one line. Here is where Peter Cole’s Bookmarklet-maker comes in, more on that later. For now save your bookmarklet and open up a new page and your favourite website there. Now click on your newly created bookmarklet…..and voila!

Peter Cole’s Bookmarklet-maker #

Peter Cole’s tool let’s us write the code in JavaScript and it encodes and writes the bookmarklet properly in a one-liner. Go over to this site and experiment with your own scripts!

Create an Internet Archive Bookmarklet to download books as images #

Enough beating around the bush, let’s get going now. Our objective was to download the books that are available only for online reading, and when we are done with the books we will delete all the files. If you visit any internet archive book’s page, and inspect it, you can see that the pages are rendered as images. What the bookmarklet does is simply iterate over all of these pages and save them as images in our local drive. Next we can visit a website and convert all of them to pdf.

  1. Find the image link that looks like the image below.
  1. Now, open the image in a new tab and click on the bookmarklet.
  1. The bookmarklet will ask you the total number of pages.
